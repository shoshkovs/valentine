# Инструкция по добавлению фотографии

1. Поместите файл с фотографией в корневую директорию проекта
2. Назовите файл `photo.jpg`
3. Рекомендуемый размер: квадратное изображение минимум 400x400 пикселей
4. Формат: JPG, PNG или GIF

Если фотография не будет добавлена, вместо неё будет отображаться сердечко ❤️
